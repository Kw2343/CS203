# CS203
Gaming hub

##to activate the back end 
go to the Command prompt select the correct folder name "backend"
Enter the command "venv\Scripts\activate"
Once you are in the virtual environment
Enter Command " py manage.py runserver"
it will populate a link for the back end


##to activate front end 
go to command prompt select the correct folder name "frontend"
enter command "npm start" 
it will start the server and you will be able to view the page
